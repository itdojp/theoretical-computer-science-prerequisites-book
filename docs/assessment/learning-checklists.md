# 学習チェックリスト

学習者が進捗を消し込むためのチェックリスト。Markdown のタスクリストとして使う。

## 全体ワークフロー

- [ ] 前提診断80問を解く。
- [ ] 採点ルーブリックで採点する。
- [ ] 弱点別リカバリールートを確認する。
- [ ] 必要章を読む。
- [ ] 章末確認チェックを解く。
- [ ] 対応する演習を解く。
- [ ] 統合到達確認テストを解く。
- [ ] ミニプロジェクトを1つ以上実装する。
- [ ] 本体教科書 readiness checklist を確認する。
