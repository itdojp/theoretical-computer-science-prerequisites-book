# 採点フォーム

## 目的

このページは、前提診断、章別レビュー、演習、統合到達確認テスト、ミニプロジェクトを一貫した形式で採点するためのフォームです。複数の受講者を扱う場合、`grading-sheet.csv` をコピーして運用してください。

## 共通採点スケール

| 点 | 判定 | 基準 |
|---:|---|---|
| 4 | 完全 | 定義、計算、証明、説明が正確。不要な飛躍がない |
| 3 | 概ね可 | 主要な考え方は正しいが、記法・境界条件・説明に軽微な不足がある |
| 2 | 部分可 | 方針は近いが、定義展開・証明・計算に重要な欠落がある |
| 1 | 不十分 | 関連概念は出ているが、結論に到達していない |
| 0 | 不可 | 未回答、または概念が誤っている |

## 統合到達確認テストフォーム

| 区分 | 問題 | 得点 | コメント |
|---|---|---:|---|

| Extended | 36-50 |  |  |
| 証明 | Q7-Q10、Q23、Q49 |  / 6 |  |
| 合計 | 1-50 |  |  |

### 判定

- 合計43/50以上、かつ証明3/6以上: 合計点と証明のゲートを達成。対象章別 readiness を確認する。
- 合計42/50以下: 失点領域を補強して再受験する。
- 証明2/6以下: 合計点に関係なく第3章を補強して再受験する。
