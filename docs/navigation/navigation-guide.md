## 推奨される読み方

1. まず [前提診断](../diagnosis.md) を解く。
2. 失点した領域を [弱点別リカバリールート](../assessment/recovery-routes.md) で確認する。
3. 対応する章を読む。各章冒頭の「この章で使う記号」で記法を先に確認する。
4. 章末の「次に読む章」から、演習・図表・Python実装へ進む。
5. 最後に [本体教科書 readiness checklist](../assessment/main-textbook-readiness.md) を使って、本体へ進むか判定する。
