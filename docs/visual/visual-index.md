# 図表ガイド

このページでは、本文で詰まりやすい概念を静的 SVG 図で補強する。Jekyll の標準 Markdown だけで表示できるように、Mermaid 依存にはしていない。

## 図表一覧

| 図 | 対象章 | 使い方 |
|---|---|---|
| 証明を書くときの基本フロー | 第3章 | 証明問題で最初に見る |
| 代表的な成長率 | 第4章 | O記法の大小比較で見る |
| 再帰呼び出しとスタック | 第5章 | 再帰と帰納法の対応を見る |
| BFSの距離層 | 第6章 | 最短距離とキューの関係を見る |
| Union-Findの代表元 | 第8章 | 集合併合の抽象化を見る |
| DFA / NFA | 第9章 | 形式言語の状態遷移を見る |
| 条件付き確率の確率木 | 第10章 | 乗法公式と全確率を確認する |
| Hamming距離 | 第12章 | 線形符号の入口で見る |
| 状態インターリーブ | 第13章 | 並行性の状態爆発を見る |
| ループ不変条件 | 第3章・第5章 | アルゴリズムの正しさ証明で見る |

## 使い方

図は本文の代替ではなく、定義を読む前後に参照する補助である。特に、図だけで正しさを主張しないこと。図で直観を得た後、定義・量化記号・証明へ戻る。
