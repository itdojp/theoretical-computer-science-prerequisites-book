## 内容上の境界

この教材は、理論計算機科学の本体教材ではありません。対象は、本体教材を読み始める前に必要となる前提知識の補強です。

含めるもの:

- 前提診断。
- 基本定義と例・非例。
- 証明・計算・実装の最小訓練。
- 本体教科書への接続。

含めないもの:

- 理論計算機科学全範囲の完全な代替説明。
- 大学院レベルの詳細な定理展開。
- 外部教材本文・図・問題の転載。

## v1.0以降の保守方針

- 外部リンク切れは代替リンクまたは注記で対応する。
