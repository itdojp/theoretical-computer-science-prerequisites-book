# 概念別逆引き

詰まり方から読む章を選ぶための逆引き。

| 詰まり方 | 読む章 | 確認観点 |
|---|---|---|
| 証明を書けない | [第1章](../part-1-core/01-sets-logic.md), [第2章](../part-1-core/02-functions-relations.md), [第3章](../part-1-core/03-proofs.md) | 定義展開、対偶、背理法、帰納法、反例を確認する。 |
| O記法を感覚で使っている | [第4章](../part-1-core/04-asymptotic-notation.md) | 定義からの証明とループ解析を確認する。 |
| 再帰が追えない | [第3章](../part-1-core/03-proofs.md), [第5章](../part-1-core/05-pseudocode-recursion.md) | 停止性、呼び出しスタック、帰納法の対応を確認する。 |
| グラフ探索が曖昧 | [第6章](../part-2-standard/06-graphs-trees.md), [第8章](../part-2-standard/08-data-structures.md) | BFS/DFS、キュー/スタック、到達可能性を確認する。 |
| 状態数の見積もりが苦手 | [第7章](../part-2-standard/07-counting.md), [第8章](../part-2-standard/08-data-structures.md), [第10章](../part-3-extended/10-probability.md) | 組合せ、状態空間、指示変数を確認する。 |
| オートマトンの記法が読めない | [第2章](../part-1-core/02-functions-relations.md), [第6章](../part-2-standard/06-graphs-trees.md), [第9章](../part-2-standard/09-formal-language-primer.md) | 関数、グラフ、DFA/NFAを確認する。 |
| 情報理論で止まる | [第7章](../part-2-standard/07-counting.md), [第10章](../part-3-extended/10-probability.md), [第12章](../part-3-extended/12-linear-algebra.md) | ビット列、確率、期待値、Hamming距離を確認する。 |
| 暗号で止まる | [第4章](../part-1-core/04-asymptotic-notation.md), [第7章](../part-2-standard/07-counting.md), [第10章](../part-3-extended/10-probability.md), [第11章](../part-3-extended/11-number-theory-algebra.md) | 計算困難性、鍵空間、合同算術、群・体を確認する。 |
| 並行計算で止まる | [第2章](../part-1-core/02-functions-relations.md), [第6章](../part-2-standard/06-graphs-trees.md), [第12章](../part-3-extended/12-linear-algebra.md), [第13章](../part-3-extended/13-concurrency-models.md) | 関係、グラフ、行列、状態遷移、安全性・活性を確認する。 |
