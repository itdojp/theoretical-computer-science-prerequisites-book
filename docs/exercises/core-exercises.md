# Core 演習

---


> **難易度ラベル**
> - **[A]** 基本: 定義を読む・単純計算をする。
> - **[B]** 標準: 複数の定義を組み合わせる。
> - **[C]** 発展: 証明・設計・実装の説明が必要。
> - **[D]** 実装: 実行可能なコードまたは明確な擬似コードが必要。
> - **[E]** 接続: 本体教科書の概念へ橋渡しする。
