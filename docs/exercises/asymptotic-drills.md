3. **[A][漸近記法]** `10n^3+n = Θ(n^3)` を示せ。
4. **[A][漸近記法]** `n = O(n log n)` を `n≥2` で説明せよ。

## C. ループ解析

13. **[B][漸近記法]** 次の計算量を求めよ。

```text
for i = 1 to n:
    constant_work()
```

14. **[B][漸近記法]** 次の計算量を求めよ。

```text
for i = 1 to n:
    for j = 1 to n:
        constant_work()
```

15. **[B][漸近記法]** 次の計算量を求めよ。

```text
for i = 1 to n:
    for j = 1 to i:
        constant_work()
```

16. **[B][漸近記法]** 次の計算量を求めよ。

```text
for i = 1 to n:
    j = 1
    while j < n:
        j = 2 * j
```

17. **[C][漸近記法]** 次の計算量を求めよ。

```text
i = n
while i > 1:
    i = floor(i / 3)
```

18. **[C][漸近記法]** 次の計算量を求めよ。

```text
for i = 1 to n:
    for j = 1 to n:
        for k = 1 to n:
            constant_work()
```

## D. 再帰式

19. **[C][漸近記法]** `T(n)=T(n/2)+1` の解を求めよ。
20. **[C][漸近記法]** `T(n)=2T(n/2)+n` の解を求めよ。
21. **[C][漸近記法]** `T(n)=T(n-1)+1` の解を求めよ。
22. **[C][漸近記法]** `T(n)=T(n-1)+n` の解を求めよ。
23. **[C][漸近記法]** `T(n)=2T(n-1)+1` は多項式時間か。理由を述べよ。

## 代表解

- 13: `Θ(n)`。
- 14: `Θ(n^2)`。
- 15: `Θ(n^2)`。総反復回数は `n(n+1)/2`。
- 16: 外側 `n` 回、内側 `Θ(log n)` 回なので `Θ(n log n)`。
- 17: `Θ(log n)`。
- 18: `Θ(n^3)`。
- 19: `Θ(log n)`。
- 20: `Θ(n log n)`。
- 21: `Θ(n)`。
- 22: `Θ(n^2)`。
- 23: 指数的。各段で呼び出しが2倍になるため、多項式時間ではない。
