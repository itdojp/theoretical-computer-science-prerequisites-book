!!! note "難易度ラベル"

### 4

### 12

`n>0` のとき `Bad(n+1)` を呼ぶため、`0` に近づかない。したがって停止しない。

### 13

`n=1` のとき `floor(1/2)=0` なので停止する。`n>1` でも再帰呼び出しごとに自然数 `n` が小さくなるため、停止する。

### 24

```python
def max_value(a):
    if not a:
        raise ValueError("empty array")
    m = a[0]
    for x in a[1:]:
        if x > m:
            m = x
    return m
```

### 25

```python
def is_sorted(a):
    for i in range(len(a) - 1):
        if a[i] > a[i + 1]:
            return False
    return True
```
