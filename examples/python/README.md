# Python examples

標準ライブラリだけで動く、理論概念の確認用コードである。

実行:

```bash
python examples/python/tests.py
```

個別実行:

```bash
python examples/python/recursion_trace.py
python examples/python/bfs_layers.py
python examples/python/dfa_simulator.py
python examples/python/state_transition.py
```

各ファイルは、小さい入力で状態変化を追えることを優先している。性能や実用性ではなく、定義と実装の対応を確認するためのコードである。
